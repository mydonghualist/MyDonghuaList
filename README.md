MyDonghuaList is a modern platform for **discovering, tracking, and managing Donghua** (Chinese animation).  
It allows users to search for series, track their watchlist, rate titles, manage their viewing progress, and engage with a community of fans.

---

## Features

- Search and discover Donghua
- Detailed pages (synopsis, genres, staff, ratings)
- Personal watchlist management
- Rate and review Donghua
- User authentication (Login & Register)
- Dark / Light theme
- Responsive design (mobile + desktop)
- Discover page — trailers, news, and community discussions
- Community posts with flairs (Trailer, News, Discussion, Fanart)
- Episode airing calendar
- Notifications

---

## Tech Stack

- **Framework:** Vue 3
- **Build Tool:** Vite
- **Language:** TypeScript
- **State Management:** Pinia
- **Routing:** Vue Router
- **Styling:** Tailwind CSS
- **Testing:** Vitest + Playwright
- **API:** MyDonghuaList Backend
